You can make the text more readable

If the text is a code you can use <code>, for example:
Test

You can represent a code in C or python using <pre> and <code>, for example: 

        
Int a, b, c;
scanf("%d %d %d", &a, &b, &c);
int soma = a + b + c;
printf("%d", soma);

You can represent a quote using <q>, exmaple:

Test

You can too move a quote, making a paragraph, like:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Quasi quisquam sit reiciendis quaerat, ad ipsa. Nihil, dolor doloremque vel mollitia obcaecati vitae accusamus sit voluptas dolorem eum, iste provident neque.

You can also indicate the meaning of a abbreviation, such as:

Thxs
